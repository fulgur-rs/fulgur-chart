fulgur-chart サンプルギャラリー

fulgur-chart が生成した静的 SVG を埋め込んだギャラリーです。対応するチャート種と 主な機能を一通り並べています。fulgur render で PDF 化すると、SVG が ベクターのまま埋め込まれます。

棒グラフ

縦棒。月次の売上推移。

横棒グラフ

indexAxis: "y" による横棒。

積み上げ棒グラフ

scales.y.stacked で系列を積み上げ。

折れ線グラフ

2 系列・tension でなめらかに。

エリアチャート

"fill": true の line。

円グラフ

スライスごとに自動配色。

ドーナツグラフ

ドーナツ。凡例は右配置。

散布図

{x, y} 点データ・線形×線形軸。

バブルチャート

{x, y, r} で第3次元を半径に。

レーダーチャート

多変量を極座標で比較。

混合チャート

dataset 別 type で棒 + 折れ線。

データラベル

plugins.datalabels で各点に値を描画。

テーマ（ダーク）

options.theme で配色を上書き。

Vega-Lite 入力

--dsl vegalite での入力例。
